# Live Journey 랜딩 페이지

실시간 여행 정보 플랫폼 라이브 저니의 랜딩 페이지입니다.

## GitHub Pages 설정

1. 저장소 Settings → Pages
2. Source: Deploy from a branch
3. Branch: `master`
4. Folder: `/landing-page` 선택
5. Save

## 로컬에서 실행

```bash
cd landing-page
python -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속
